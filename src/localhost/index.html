What is nYume?

nYume is a small and easy, but fully functional HTTP-server. It was written in Freepascal using standart cross-platform lybraries. nYume can be compiled under all platforms supported by Freepascal compiler.

nYume is just an experiment which can show all power of language Freepascal. nYume is cross-platform, multithreading, very small application. Source code of nYume consists of 4 files common length less than 1000 lines. It can be compiled in one second using one command. Binary program is really small and works fast.

How to use nYume?

  • As a base platform for local applications with Web-interface.
  • As a platform for network applications.
  • As a LAN HTTP-server.
  • As a HTTP-server on your home computer.

How not to use nYume?

  • As a HTTP-server with dozens of complicated sites with dynamic content.
  • As a HTTP-server for hosting providers.
  • As a platform for testing cross-server Web-applications.

Advantages of nYume

  • Small size.
  • Cross-platform.
  • Multithreading.
  • Easy in installation.
  • Easy in configuration.
  • Possible to run CGI applications.
  • Possible to organize virtual hosting with a set of sites on one IP.
  • Opensource.
